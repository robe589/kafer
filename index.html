How to participate

  1. Fork this repository
  2. Solve the problem
  3. Create README.md on each problem directory with YOUR_NAME.md
  4. Append a link to YOUR_NAME.md in SUMMARY.md
  5. Create a link to your repository of the problem in YOUR_NAME.md
  6. Write what you did, especially paid for attention what to do, not to do into YOUR_NAME.md
  7. Create a PR :)
